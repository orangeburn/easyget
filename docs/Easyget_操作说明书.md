# Easyget 操作说明书

## 1. 这是什么

Easyget 是一个帮助你搜集招标、采购、公示信息的工具。

你只需要告诉它：

- 你想找什么关键词
- 你想重点关注哪些网站
- 你想监控哪些公众号

系统会自动去搜集信息，并把结果整理成列表，方便你快速筛选。

它的定位不是替你做最终判断，而是帮你节省查找和整理信息的时间。

---

## 2. 它能做什么

当前版本主要支持这些功能：

- 按关键词进行全网搜索
- 按指定网址进行定向监控
- 按微信公众号名称进行搜集
- 按地区、金额、发布时间做筛选
- 使用模型自动扩展关键词
- 实时刷新线索，不需要等全部任务结束
- 对线索做收藏、忽略、过滤分类
- 导出已选线索为 `CSV` 文件
- 保存系统设置和任务配置

---

## 3. 使用前需要准备什么

### 电脑环境

- Python
- Node.js
- npm

### 建议提前准备的信息

- 你要搜索的关键词
- 你关注的地区
- 你可接受的最低项目金额
- 你想监控的网站地址
- 你想关注的公众号名称
- 如果要启用模型或第三方搜索，需要对应的 API Key

### 注意事项

- `API Key` 相当于密码，请妥善保管。
- 不要把包含真实 `API Key` 的截图发到群里或公开平台。
- 如果暂时没有模型 API 和第三方搜索 API，也可以先使用部分能力。

## 4. 启动方式

建议先启动后端，再启动前端。

### 启动后端

在 PowerShell 中进入项目目录后执行：

```powershell
cd backend
python -m pip install -r requirements.txt -t .deps
$env:PYTHONPATH = (Resolve-Path .deps).Path
python run.py
```

默认情况下，后端地址是：

```text
http://127.0.0.1:8000
```

### 启动前端

打开另一个 PowerShell 窗口，执行：

```powershell
cd frontend
npm install
npm run dev
```

前端启动后，通常会提供一个本地地址，例如：

```text
http://localhost:5173
```

用浏览器打开即可。

## 5. 页面说明

### 1. 任务配置

用于填写搜索条件，例如关键词、目标网站、公众号、地区和金额等。

### 2. 线索情报

用于查看系统搜集到的线索，并进行收藏、忽略、查看详情等操作。

### 3. 系统设置

用于填写模型 API、搜索 API 等系统级配置。

### 4. 顶部状态栏

可以查看当前策略、任务状态，并快速进入任务配置或系统设置。

---

## 6. 推荐使用顺序

第一次使用时，建议按下面顺序进行：

1. 启动后端和前端
2. 进入“系统设置”
3. 填写并保存系统设置
4. 进入“任务配置”
5. 填写关键词、地区、金额等条件
6. 启动任务
7. 到“线索情报”查看结果
8. 收藏、忽略、导出需要的内容

---

## 7. 系统设置怎么填

### 1. 模型 API

作用：

- 关键词扩展
- 标题语义过滤
- 一部分智能判断

你会看到这些字段：

- `模型 API Key`
- `模型 Base URL`
- `模型名称`

填写建议：

- 如果你有模型服务，就打开开关并填写完整信息
- 如果暂时没有，可以先关闭

注意：

- `Base URL` 需要以 `http://` 或 `https://` 开头
- 建议先点“测试连接”，确认可用后再保存

### 2. 搜索 API

目前支持：

- `Serper`
- `Tavily`

作用：

- 补充更多搜索结果

填写建议：

- 有哪个 Key 就启用哪个
- 没有可以先不启用

额外说明：

- `Tavily API Key` 一般以 `tvly-` 开头

### 3. 本地浏览器搜索

这是系统内置的搜索能力：

- 默认开启
- 不能手动关闭

即使你没有第三方搜索 API，系统也仍然可以通过本地浏览器搜索一部分信息。

## 8. 任务配置怎么填

### 1. 核心搜索词

这是最重要的字段。

例如：

```text
广州 智慧医院 软件开发
```

建议把下面几类词组合起来：

- 地点
- 行业
- 需求类型

### 2. LLM 扩词

这一部分由系统自动生成，用于补充相关关键词。

如果模型 API 可用，系统会自动扩展；如果模型不可用，则会使用降级策略。

### 3. 目标省份 / 目标城市

用于限定地区范围。

- 如果全国都可以看，选“全国”
- 如果只关注某个省市，就按实际情况选择

### 4. 最低金额

例如填写 `50`，表示更关注金额不低于 50 万的项目。

### 5. 发布时间

可以选择：

- 不限
- 三天内
- 一周内
- 一个月内

如果你更关注新信息，建议优先选择时间较短的范围。

### 6. 搜集频率

支持：

- `自动循环`
- `每一天`

一般来说：

- 想持续盯新消息时，用 `自动循环`
- 想每天固定执行一次时，用 `执行一次`

### 7. 定向监控网址

一行填写一个网址。

例如：

```text
https://www.ccgp.gov.cn/
https://www.gzggzy.cn/
```

说明：

- 占位示例不会自动加入任务
- 只有你实际填写的 URL 才会进入定向爬虫
- 部分站点有较强反爬机制，可能抓取失败

### 8. 微信公众号

一行填写一个公众号名称。

例如：

```text
某某招标中心
某某采购平台
```

### 9. 启动任务

填写完成后点击：

- 首次使用时一般是 `开始全网搜集`
- 在仪表盘中修改时一般是 `应用修改`

系统会启动任务，并跳转到线索列表页面。

## 9. 常见使用方式

### 方式 1：只用关键词搜索

适合还不知道重点网站，只想先做大范围搜索。

### 方式 2：只盯固定网站

适合你已经知道哪些官网最重要。

### 方式 3：关键词 + 网址一起用

通常效果更好。

因为既能全网找，也能重点盯住关键来源。

---

## 10. 线索情报页面怎么用

页面中常见分类有：

- `待处理`
- `已收藏`
- `已忽略`
- `已过滤`

其中：

- `待处理` 是建议优先查看的内容
- `已收藏` 是你判断为有价值的线索
- `已忽略` 是你确认不需要的内容
- `已过滤` 是系统规则或模型判定不符合条件的内容

---

## 11. 每条线索可以做什么

### 1. 收藏

适合需要继续跟进的项目。

### 2. 忽略

适合明显无关的内容。

### 3. 查看详情

打开右侧详情面板，查看：

- 项目金额
- 项目地点
- 截止时间
- 来源链接
- 内容概要
- 网页正文或抓取结果

### 4. 查看原文

可以直接打开原网页。

### 5. 导出

支持勾选多条线索并导出为 `CSV` 文件，方便用 Excel 打开。

---

## 12. 关键词怎么写更有效

推荐写法：

```text
深圳 医疗信息化 平台建设
广州 学校 信息系统 运维
江苏 医院 数据治理 项目
```

不推荐只写非常宽泛的词，例如：

```text
软件
项目
医院
```

一个简单原则：

`地点 + 行业 + 事情`

例如：

```text
北京 医疗 软件开发
```

## 13. 没结果时先检查什么

### 页面打不开

请检查：

- 后端是否已启动
- 前端是否已启动
- 打开的地址是否正确

### 点击开始没反应

请检查：

- 是否至少填写了一项采集条件
- 后端是否仍在运行

### 系统设置保存失败

请检查：

- 后端服务是否正常
- API Key 是否为空
- Base URL 是否填写正确

### 测试连接失败

请检查：

- 网络是否正常
- API Key 是否正确
- 模型地址是否可访问
- Tavily Key 是否符合格式

### 线索太少

可以尝试：

- 调整关键词
- 放宽地区限制
- 放宽金额限制
- 放宽发布时间限制
- 增加定向网址
- 增加公众号来源

### 线索太多

可以尝试：

- 把关键词写得更具体
- 增加地区限制
- 提高最低金额
- 缩短发布时间范围

## 14. 一次完整示例

假设你想找“广州医院软件项目”，可以这样操作：

1. 打开系统设置
2. 填好模型 API 或搜索 API
3. 点击“测试连接”
4. 点击“保存设置”
5. 打开任务配置
6. 输入 `广州 智慧医院 软件开发`
7. 选择 `广东` 和 `广州`
8. 最低金额填写 `50`
9. 发布时间选择 `一个月内`
10. 点击“开始全网搜集”
11. 查看线索情报
12. 对有价值的内容进行收藏、忽略或导出
